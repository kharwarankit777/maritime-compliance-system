# Maritime Compliance System

## Overview
A full-stack maritime compliance management system designed to manage ship maintenance tasks, safety drills, and compliance tracking.

## Features
- JWT Authentication
- Role-based access
- Maintenance task management
- Safety drill management
- Compliance dashboard
- Analytics overview

## Tech Stack

### Frontend
- React.js
- Axios
- Context API

### Backend
- Spring Boot
- Spring Security
- JWT Authentication
- Hibernate / JPA

### Database
- MySQL

## Setup Instructions

### Backend
1. Configure MySQL in application.properties
2. Run:
   mvn spring-boot:run

### Frontend
1. Install dependencies:
   npm install

2. Start React app:
   npm start
